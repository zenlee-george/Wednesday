There are different programming languages that involve web development

HTML (Hypertext Markup Language)

HTML is responsible for creating the structure and content of web pages. It defines the elements and layout of a webpage, such as headings, paragraphs, images, and links.

CSS (Cascading Style Sheets)

CSS is responsible for styling the appearance of web pages. It controls the layout, colors, fonts, and other visual aspects of HTML elements, allowing for the customization and design of web pages.

JavaScript

JavaScript is responsible for adding interactivity and dynamic behavior to web pages. It enables features such as form validation, animations, interactive maps, and responsive user interfaces.
